# Pets

Pet definitions and sprite assets used by the desktop pet engine.

## Layout

- `eSheep-modern/` - modern JSON-based pet definition and sprites.
- `eSheep-modern-debug/` - debug-oriented sprite and animation assets.
- `esheep64/` - legacy XML-based pet definition.

## Notes

Each pet directory is self-contained. Modern pets use `animations.json`; legacy pets use `animations.xml`.

Keep large generated previews or exports out of version control unless they are part of the pet definition itself.
