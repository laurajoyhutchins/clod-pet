# Public Assets

Static HTML and CSS entrypoints for Electron windows live here.

## Files

- `index.html` - main app shell.
- `chat.html` - chat window.
- `control-panel.html` - settings panel.
- `pet.html` - pet window.
- `editor.html` - editor shell.
- `control-panel-*.css` - control-panel styling variants.
- `editor.css` - editor styling.

## Notes

- These files are copied into the app build output as-is.
- Keep them free of runtime TypeScript dependencies.
